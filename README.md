# 浜名湖うなぎサイト「鰻処 湖松（こまつ）」

浜名湖うなぎ専門店をイメージした、和モダンなランディングページのデモです。

---

## ⚠ 重要：これは架空のデモサイトです

**本サイトに登場する店舗・情報はすべて架空のものです。実在する店舗・団体・商品とは一切関係ありません。**

- 店名「鰻処 湖松」は架空の名称です。
- 住所・電話番号・営業時間・価格・実績等の数値は、すべてサンプルとして用意した架空の内容です。
- お取り寄せの「カート」やご予約フォームは画面上の**デモ動作のみ**で、実際の注文・予約・決済・データ送信は一切行われません。
- お客様の声などのレビューも創作であり、実在の人物の発言ではありません。

実在の店舗等でご利用になる場合は、必ず正しい情報に差し替えてください。

---

## 内容

- `index.html` — サイト本体（CSS・JavaScript を内包した自己完結ファイル）
- `styles.css` — スタイルシート（`index.html` に内包済みのため、現状は参考用）
- `script.js` — スクリプト（`index.html` に内包済みのため、現状は参考用）

## 主な機能・セクション

- ヒーロー（炭火・蒲焼をイメージした背景アニメーション）
- こだわり紹介 / 実績カウンター（スクロール時にアニメーション）
- お品書き / お取り寄せ（カート機能：追加・削除・合計計算）
- お客様の声 / 店舗案内（地図埋め込み）/ ご予約フォーム
- レスポンシブ対応（スマートフォンではハンバーガーメニュー）

## 表示方法

`index.html` をブラウザで開くだけで表示できます（外部ファイルに依存しない自己完結構成）。

## 技術

- HTML / CSS / バニラ JavaScript のみ。ビルド不要。
- フォント：Shippori Mincho、Noto Sans JP（Google Fonts）
